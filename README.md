# Automation-and-Webscraping-Using-Selenium-And-Chrome-Webdriver
This project is one part of a bigger project where a machine learning model is utilized to predict salary ranges for various engineering job titles. In this specific project selenium and chrome webdriver is used to automate the login and browsing of linkedIn job profiles while webscraping key job infos. The project was carried out as follows:
* Creating Selenium webdriver instance.
* Opening chrome and navigating to linkedin web page.
* Automating linkedin login and navigating to engineering jobs page.
* Webscraping job titles, job location, company name, salary and other benefits.
* Storing the info in a python dictionary and later converting it to a pandas dataframe.
* And finally saving the data in csv format.
